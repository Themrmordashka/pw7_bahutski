Explain "repaint" and "reflow" of the webpage.

Repaint: When changes in CSS styles are applied to an element that do not affect its layout, only the affected part of the webpage is repainted without changing the layout of other elements. Reflow: When changes in CSS styles or DOM structure occur that affect the layout of the webpage, causing elements to be repositioned or resized, a reflow is triggered, which is more costly in terms of performance.

How does JavaScript on the webpage affects the page load speed and performance?

JavaScript execution can block rendering and make the page load slower. Poorly optimized JavaScript code can also lead to higher CPU and memory usage, affecting page performance negatively. However, using JavaScript to enhance user experience through features like lazy loading, asynchronous loading, and caching can improve overall performance.

What is "minification" of the script? Why it is useful to minify your scripts?

Minification is the process of removing unnecessary characters from code, such as whitespace, comments, and line breaks, to reduce its size. Minifying scripts helps improve page load times by reducing file size, as smaller files can be downloaded faster by the browser. It also reduces bandwidth usage and improves website performance.

What is a JS Web Worker and when do we need it?

A JavaScript Web Worker is a separate thread that runs concurrently with the main JavaScript thread, allowing for parallel execution of code. Web Workers are useful for tasks that require intensive computation or long-running operations, such as data processing, image manipulation, and background tasks, without blocking the main UI thread, thus improving responsiveness and performance of web applications.
